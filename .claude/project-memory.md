# Project environment

- Expo SDK 57.0.4, React Native 0.86.0, React 19.2.3, New Architecture and React Compiler enabled.
- Yarn 4.12.0 with Node 22.22.3.
- Start Metro with `yarn start` on port 8081.
- Run native apps with `yarn ios` or `yarn android`.
- iOS workspace: `ios/bare.xcworkspace`; scheme: `bare`; bundle ID: `com.tovtam.bare`.
- Validate with `yarn typecheck` and `yarn lint`.
- Navigation uses React Navigation 7 static configuration; styling uses Uniwind 1.10 with Tailwind CSS 4.
- The mobile application is for staff and administrators; customers use a separate web application.
